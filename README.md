# PVCloudGroupn
An online shopping mall(在线商城，持续迭代中..)

# 特别说明  
*该框架的主要架构为retrofit+dagger2+rxjava+gson
*该项目最基础的原型来自菜鸟窝，我在其的一个版本上进行改编的，其中大部分的基础工具使用的是菜鸟窝整理的，特别感谢他们的开源
